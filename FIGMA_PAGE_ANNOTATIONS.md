## 4. 交互流程图

### 4.1 用户流程

```
Landing Page
    ↓
[连接钱包] → 钱包选择弹窗 → 选择钱包 → 签名
    ↓
Portal Page
    ↓
[首次进入] → 推荐绑定弹窗 → 确认/跳过
    ↓
选择应用
    ├→ [D3-AI] → D3-AI分析站
    └→ [D3-Fi] → D3-Fi应用
```

### 4.2 D3-Fi 内部流程

```
Dashboard
    ↓
├→ [资产] → Assets 页面
├→ [入场] → Enter 页面 → 选择方式 → 输入金额 → 确认交易
├→ [投票] → Vote 页面 → 选择Gauge → 分配投票权 → 确认
├→ [贿赂市场] → Bribe 页面 → 查看贿赂池 → 参与贿赂
├→ [分红] → Dividends 页面 → 查看分红 → 领取
├→ [推荐管理] → Network 页面 → 查看推荐链接 → 复制/分享
├→ [安全机制] → Safety 页面 → 查看守护机制
└→ [帮助] → Help 页面 → 查看FAQ
```
