## 1. 按钮组件库

### 1.1 D3Button - Primary（主按钮）

**尺寸变体**
| 尺寸 | 高度 | 横向内边距 | 字体大小 | 用途 |
|------|------|----------|---------|------|
| Small | 32px | 12px | 14px | 导航栏、卡片操作 |
| Medium | 40px | 16px | 16px | 表单、常规操作 |
| Large | 48px | 32px | 18px | Hero区、主CTA |

**样式参数**
- **背景**：线性渐变 135°
  - 起点：oklch(0.35 0.18 350) [深酒红]
  - 终点：oklch(0.45 0.16 350) [浅酒红]
- **文字颜色**：oklch(1 0 0) [白色]
- **字体**：Noto Sans SC 600
- **圆角**：8px
- **边框**：无
- **阴影**：
  - 默认：0 2px 8px rgba(107, 27, 71, 0.1)
  - Hover：0 8px 16px rgba(107, 27, 71, 0.2)
  - Active：0 4px 12px rgba(107, 27, 71, 0.15)

**交互状态**
- **Default**：不透明度100%
- **Hover**：阴影增强，亮度+3%
- **Active**：scale 97%，阴影减弱
- **Disabled**：不透明度50%，cursor: not-allowed

**Figma设置**
```
Component: Button/Primary
Variants:
  - Size: Small | Medium | Large
  - State: Default | Hover | Active | Disabled
```

## 2. 卡片组件库

### 2.1 DataCard（数据卡）

**结构**
```
┌─────────────────────┐
│  标题 (H4)          │
│  ─────────────────  │
│  数值 (H2)          │
│  单位说明 (Small)   │
└─────────────────────┘
```

**样式参数**
- **背景**：oklch(1 0 0) [白色]
- **边框**：1px solid oklch(0.68 0.12 70 / 0.2) [金色/20%]
- **圆角**：12px
- **内边距**：24px
- **阴影**：0 2px 8px rgba(0, 0, 0, 0.08)
- **Hover阴影**：0 8px 16px rgba(0, 0, 0, 0.12)

**间距**
- 标题到数值：8px
- 数值到单位：4px

**字体**
- 标题：14px 400 oklch(0.55 0.02 280) [灰色]
- 数值：32px 700 oklch(0.35 0.18 350) [深酒红]
- 单位：12px 400 oklch(0.55 0.02 280) [灰色]

**变体**
- **Elevated**：阴影更强，适合突出显示
- **Flat**：无阴影，用于密集布局
- **Hover**：阴影增强，微微上浮（transform: translateY(-2px)）

## 14. 在Figma中实现

### 14.1 创建步骤

1. **创建颜色样式**
   - 在Assets面板中创建所有颜色变量
   - 命名规范：Color/Primary-700、Color/Gold-700等

2. **创建字体样式**
   - 创建所有字体大小和权重组合
   - 命名规范：Typography/H1、Typography/Body等

3. **创建组件**
   - 为每个组件创建主组件
   - 添加变体（Size、State等）
   - 使用颜色和字体样式

4. **组织结构**
   ```
   Components/
   ├── Button/
   │   ├── Primary
   │   ├── Secondary
   │   └── Outline
   ├── Card/
   │   ├── Data
   │   ├── Premium
   │   └── Comparison
   ├── Badge/
   ├── Input/
   ├── Table/
   ├── Navigation/
   └── ...
   ```

5. **创建页面**
   - Landing Page
   - Portal Page
   - D3-Fi Dashboard
   - D3-Fi Assets
   - ...（其他页面）
